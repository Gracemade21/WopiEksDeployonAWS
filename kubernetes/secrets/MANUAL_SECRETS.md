# WOPI Kubernetes Secrets

These secrets are created automatically by Terraform. This file documents what they contain.

## Secrets Created by Terraform

### wopi-database-secret
- **host**: Database hostname
- **port**: Database port (1433)
- **username**: Database username (admin)
- **password**: Database password (auto-generated by AWS)
- **database**: Database name (wopi_database)
- **endpoint**: Full endpoint
- **connection_string**: Full SQL Server connection string

### wopi-redis-secret
- **redis_host**: Redis primary endpoint
- **redis_port**: Redis port (6379)
- **redis_endpoint**: Redis configuration endpoint (for cluster mode)
- **redis_connection_string**: Full Redis connection string

## Manual Secret Creation (if needed)

If you need to create these secrets manually:
```bash
# Create database secret
kubectl create secret generic wopi-database-secret \
  -n wopi-services \
  --from-literal=host=<db-host> \
  --from-literal=port=1433 \
  --from-literal=username=admin \
  --from-literal=password=<db-password> \
  --from-literal=database=wopi_database \
  --from-literal=endpoint=<db-endpoint> \
  --from-literal=connection_string="Server=<host>,1433;Database=wopi_database;User Id=admin;Password=<pwd>;TrustServerCertificate=true;"

# Create redis secret
kubectl create secret generic wopi-redis-secret \
  -n wopi-services \
  --from-literal=redis_host=<redis-host> \
  --from-literal=redis_port=6379 \
  --from-literal=redis_endpoint=<redis-endpoint> \
  --from-literal=redis_connection_string=<redis-host>:6379
```
